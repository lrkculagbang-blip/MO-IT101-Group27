# MotorPH Payroll System

## 📌 Description
This is a Java-based payroll system that computes employee weekly salary, attendance, and deductions using CSV data.

### Classes and Their Purpose:

- **Employee.java**
  - Stores employee information (name, ID, salary, etc.)

- **Attendance.java**
  - Handles working hours and computes total weekly hours

- **Payroll.java**
  - Calculates salary, deductions (SSS, PhilHealth, Pag-IBIG), and net pay

- **Main.java**
  - Controls the program flow and handles user input

---

## 🎯 Why Use Multiple Classes?

- Separates responsibilities (each class has one job)
- Makes the code easier to understand
- Easier to fix and update
- Follows real-world programming practices

---

## ⚙ System Logic

- Week 1–3: Salary is computed without deductions  
- Week 4: All monthly deductions are applied  

---

## ▶ How to Run

1. Open the project in NetBeans
2. Run `Main.java`
3. Enter Employee ID
4. Enter Week (1–4)
5. View payroll output
